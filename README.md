# 🌦️ Weather App (PyQt5)

A simple desktop weather application built with **PyQt5** that allows users to enter a city name and retrieve real-time weather data using the OpenWeatherMap API.

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Displays temperature in Celsius  
- 📝 Shows weather description  
- ⚠️ Handles API and network errors gracefully  
- 🎨 Clean and responsive GUI using PyQt5  

---

## 🛠️ Technologies Used

- Python  
- PyQt5  
- Requests  
- OpenWeatherMap API
